# The recommended shape from GRoLTS criteria

The recommended shape from GRoLTS criteria

## Usage

``` r
grolts_recommended(x)
```

## Arguments

- x:

  A `gbtm_criteria` object from
  [`apply_grolts_criteria()`](https://fabregithub.github.io/gbtmkit/reference/apply_grolts_criteria.md).

## Value

A one-row data frame (the top-ranked surviving shape), or `NULL` if none
qualified.
