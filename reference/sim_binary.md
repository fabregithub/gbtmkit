# Synthetic binary trajectory data

A simulated dataset with four planted latent groups, for demonstrating
and testing binary (LOGIT / `family = "binomial"`) group-based
trajectory models. It is entirely synthetic and domain-neutral; the true
group labels are included so recovery can be checked.

## Usage

``` r
sim_binary
```

## Format

A data frame with 1500 rows and 24 columns:

- id:

  Subject identifier.

- x1, x2:

  Covariates, present but not driving the outcome.

- y1, y2, y3, y4, y5, y6, y7, y8, y9, y10:

  Binary outcomes at occasions 1–10.

- t1, t2, t3, t4, t5, t6, t7, t8, t9, t10:

  Occasion times.

- true_group:

  Ground-truth group label (1–4); not used in fitting.

## Details

The four groups follow distinct log-odds trajectories over ten
occasions, mixing polynomial orders: linear rising (group 1), cubic
rise-peak-decline (group 2), cubic decline-trough-recovery (group 3),
and linear falling (group 4). Generated by `data-raw/simulate-data.R`
with a fixed seed.
