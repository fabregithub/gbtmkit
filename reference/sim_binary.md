# Synthetic binary trajectory data

A simulated dataset with four planted latent groups, for demonstrating
and testing binary (LOGIT / `family = "binomial"`) group-based
trajectory models. It is entirely synthetic and domain-neutral; the true
group labels are included so recovery can be checked.

## Usage

``` r
sim_binary
```

## Format

A data frame with 1500 rows and 12 columns:

- id:

  Subject identifier.

- x1, x2:

  Covariates, present but not driving the outcome.

- y1, y2, y3, y4:

  Binary outcomes at occasions 1–4.

- t1, t2, t3, t4:

  Occasion times.

- true_group:

  Ground-truth group label (1–4); not used in fitting.

## Details

The four groups follow distinct log-odds trajectories over four
occasions: stable-high, falling, rising, and stable-low. Group
separation is wide because binary data over few occasions is
low-information. Generated by `data-raw/simulate-data.R` with a fixed
seed.
