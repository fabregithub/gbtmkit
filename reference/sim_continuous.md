# Synthetic continuous trajectory data

A simulated dataset with four planted latent groups, for demonstrating
and testing continuous (CNORM / `family = "gaussian"`) group-based
trajectory models. Entirely synthetic and domain-neutral, with true
group labels included so recovery can be checked.

## Usage

``` r
sim_continuous
```

## Format

A data frame with 1200 rows and 12 columns:

- id:

  Subject identifier.

- x1, x2:

  Covariates, present but not driving the outcome.

- y1, y2, y3, y4:

  Continuous outcomes at occasions 1–4.

- t1, t2, t3, t4:

  Occasion times.

- true_group:

  Ground-truth group label (1–4); not used in fitting.

## Details

The four groups follow distinct mean trajectories over four occasions:
stable-high, falling, rising, and stable-low, with Gaussian within-group
noise. Separation is generous relative to the noise so the single-start
CNORM fit avoids local optima. Generated by `data-raw/simulate-data.R`
with a fixed seed.
