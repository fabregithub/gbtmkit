# Synthetic continuous trajectory data

A simulated dataset with four planted latent groups, for demonstrating
and testing continuous (CNORM / `family = "gaussian"`) group-based
trajectory models. Entirely synthetic and domain-neutral, with true
group labels included so recovery can be checked.

## Usage

``` r
sim_continuous
```

## Format

A data frame with 1200 rows and 24 columns:

- id:

  Subject identifier.

- x1, x2:

  Covariates, present but not driving the outcome.

- y1, y2, y3, y4, y5, y6, y7, y8, y9, y10:

  Continuous outcomes at occasions 1–10.

- t1, t2, t3, t4, t5, t6, t7, t8, t9, t10:

  Occasion times.

- true_group:

  Ground-truth group label (1–4); not used in fitting.

## Details

The four groups follow distinct mean trajectories over ten occasions,
mixing polynomial orders: linear rising (group 1), cubic
rise-peak-decline (group 2), cubic decline-trough-recovery (group 3),
and linear falling (group 4), with Gaussian within-group noise.
Separation is generous relative to the noise so the single-start CNORM
fit avoids local optima. Generated by `data-raw/simulate-data.R` with a
fixed seed.
